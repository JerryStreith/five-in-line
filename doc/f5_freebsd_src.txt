1. Поставить wget
Если команды wget нет, поставить этот порт:
cd /usr/ports/ftp/wget
make install

2. Поставить boost
cd /usr/ports/devel/boost-libs
make install

3. Скачать, распаковать.
wget http://visicar.com/download/f5_src_0820.tar.gz
tar -xzf f5_src_0820.tar.gz

2. Указать переменные окружения
	Для tcsh
  setenv BOOST /usr/local/include
	
	Для bash
  BOOST=/usr/local/include; export BOOST

3.  собрать
cd f5/solver/
make -f Makefile.nolib
cd ..
mkdir spawn
cp solver/solver spawn/
cp solver/solve.sh spawn/
cp f5_spawn.sh spawn/
cp f5_solve.sh spawn/


4. Запуск
cd spawn

можно просто запустить из консоли:
./solve.sh

можно запустить так, чтобы можно было закрыть консоль а процесс остался:
nohup ./solve.sh

отредактировать файл f5_spawn.sh (по умолчанию 2 процесса)
в строчке
while [ $c -lt 2 ]
поставить чило процессов. Больше 2х не рекомендую, чтобы не перегружать 
боевой сервак посторонними задачами

./f5_spawn.sh

Когда надоест, мочить надо процессы  f5_solve.sh. В данном случае 17840 17845

В топе будет стоять solver. Его мочить бесполезно. Т.к. вместо него стартанёт 
другой.

Останавливать процесс можно в любой момент.

Пожалйуста не меняйте остальные параметры!
